# Websearch 0.8.18

Webseite mit deinem Lieblings-Suchanbieter durchsuchen.

<p align="center"><img src="websearch-screenshot.png?raw=true" alt="Screenshot"></p>

## Wie man eine Websuche verwendet

Erstelle eine `[websearch]`-Abkürzung. Es wird nach Seiten gesucht, die auf deiner Webseite enthalten sind. Sofern etwas gefunden wurde, werden die Ergebnisse auf einer Seite deines bevorzugten Suchanbieters angezeigt. 

Das folgende optionale Argument ist verfügbar: 

`site` = Überschreibe die zu durchsuchende Webseite (Standard: aktuelle Website-Adresse). 

## Beispiele

Webseite durchsuchen: 

    Kaffee
    Milch zucker
    "Milch und Zucker"

Suchfeld anzeigen:

    [websearch]

Suchfeld für eine andere Webseite anzeigen: 

    [websearch datenstrom.se]

## Installation

[Erweiterung herunterladen](https://github.com/datenstrom/yellow-extensions/raw/master/zip/websearch.zip) und die Zip-Datei in dein `system/extensions`-Verzeichnis kopieren. Rechtsklick bei Safari.

Basierend auf [einem Pen erstellt auf CodePen.io](https://codepen.io/pagedart/pen/VwvzEbx), freigegeben unter der MITLizenz.
